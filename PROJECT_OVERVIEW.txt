================================================================================
                              WHAT IS THIS?
================================================================================

RefLoop Bot is a fully functional Telegram bot that enables users to:
- Submit referral links (pay 15 Stars OR complete 3 verified claims)
- Browse available referral links by category
- Claim rewards by completing sign-ups
- Earn Telegram Stars (1 Star per verified claim after the first 3)
- Track progress and unlock free submissions

================================================================================
                            KEY FEATURES
================================================================================

‚úÖ Dual Submission Model
   - Pay 15 Telegram Stars for immediate submission
   - OR complete 3 verified claims to unlock FREE submission

‚úÖ Progressive Reward System
   - Claims 1-3: No Stars, but track progress
   - Claim 3: Unlock 1 free link submission
   - Claims 4+: Earn 1 Telegram Star per verified claim

‚úÖ Screenshot Verification
   - All claims require screenshot proof
   - Manual admin approval workflow
   - Prevents fraud and abuse

‚úÖ Telegram Stars Integration
   - Native Telegram payment system
   - 15 Stars for link submission
   - 1 Star rewards for verified claims

‚úÖ Category System
   - 10 predefined categories
   - Easy browsing and discovery
   - Organized link management

‚úÖ Security Features
   - Username validation (public username required)
   - Admin authorization
   - Duplicate claim prevention
   - SQL injection prevention
   - Max 5 claims per link
